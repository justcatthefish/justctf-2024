# justCTF 2024 CTFs

This repo contains sources for [justCTF 2024 teaser](./teaser/) and [justCTF 2024 onsite finals](./finals/) competitions hosted by [justCatTheFish](https://ctftime.org/team/33893).  

The [justCTF 2024 onsite finals](./finals/) were organized on HackYeah event in Cracow which was the biggest stationary hackathon in Europe.

You can find their respective challenge sources and summaries in subdirectories.
